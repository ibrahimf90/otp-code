# 🔐 OTP Verification UI

A sleek, animated, and fully accessible **One-Time Password (OTP) verification** page built with vanilla HTML, CSS, and JavaScript. Features a glassmorphism design, multi-language support, and a complete set of UX enhancements.

## ✨ Features

🎨 **Glassmorphism UI** | Frosted-glass card with animated gradient orbs in the background |
🌐 **Multi-language** | Full support for **English**, **German (Deutsch)**, and **Arabic (عربي)** including RTL layout |
📱 **Phone Masking** | Masked phone number displayed in subtitle (e.g. `+1 •••• •• 89`) |
🔢 **Paste Support** | Paste a full OTP code — boxes auto-fill and auto-submit |
⏱️ **Resend Cooldown** | 30-second countdown before resend is available again |
🔒 **Attempt Limiter** | Locks the form for 60 seconds after 3 failed attempts |
🔴 **Attempt Dots** | Visual red dot indicators track each failed attempt |
✅ **Animated Checkmark** | Smooth SVG draw-on animation on successful verification |
✅ **Correct Code** | "1234".

## 🛠️ Tech Stack

- **HTML5** — Semantic markup, `autocomplete="one-time-code"` for mobile OTP autofill
- **CSS3** — Custom properties, `backdrop-filter`, keyframe animations, SVG stroke animation
- **Vanilla JavaScript** — Zero dependencies, ES6+
